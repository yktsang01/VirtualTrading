All ISO data come from ISO-3166 (country codes) and ISO-4217 (currency codes). These are publicly available on the internet. The official source is from ISO.org. An easy reference is provided by IBAN.com.

ISO source also provides the currency minor units (the number of decimal places). This minor units shall be useful for formatting and display purposes. If none is provided, it will default to 2.
